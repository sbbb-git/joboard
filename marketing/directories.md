## Standard submission template

Most directories ask for some subset of: name, tagline, description,
categories, logo, screenshots, founder name, email.

**Tagline (60 chars)**
```
Free remote tech job board, refreshed daily, in 7 languages
```

**Short description (160 chars)**
```
slateremote.com aggregates remote tech jobs from 8 public job board APIs into one fast searchable index. Free, no signup, 7 languages, refreshed daily.
```

**Long description (500 chars)**
```
slateremote.com is a free aggregator of remote tech jobs pulled from the public APIs of eight job boards (Remotive, RemoteOK, We Work Remotely, Arbeitnow, Himalayas, Jobicy, The Muse, HN Who's Hiring). The full index is regenerated daily and served as a static site. Filter by role, country, seniority, employment type, and salary band, all client-side without an account. Available in 7 languages with proper hreflang, native long-form guides on landing and sustaining a remote tech career.
```

**Categories**: Tech Hiring, Remote Work, Job Search, HR Tech, Recruiting

**Tags**: remote jobs, tech jobs, developer jobs, job aggregator, free job board, no signup, multilingual
